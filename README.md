# Starsand Island Korean Patch

비공식 스타샌드 아일랜드 한글패치 배포 저장소입니다.

## 최신 배포본

- 패치 버전: `v1.0.15`
- Xbox PC 기준 확인 게임 버전: `2026.16.54.0`
- Steam 기준 확인 buildid: `22987508`
- 지원 플랫폼: `Steam / Xbox PC`

최신 배포 파일은 Releases 탭에서 받을 수 있습니다.

- 최신 릴리스: https://github.com/yuniwon/starsand-island-korean-patch/releases/latest

## 설치 방법

1. 최신 `StarsandIsland_KoreanPatch_*.zip`을 다운로드합니다.
2. 압축을 원하는 위치에 풉니다.
3. `스타샌드 아일랜드 한글패치 런처.exe`를 실행합니다.
4. 런처가 Steam/Xbox 게임 경로를 자동으로 찾습니다.
5. 자동 탐지에 실패하면 런처에서 게임 `Content` 폴더 또는 게임 루트 폴더를 직접 선택합니다.
6. `설치/업데이트`를 누릅니다.
7. 설치 후 게임 내 언어를 `English`로 선택하면 한글이 적용됩니다.

`Start_Korean_Patch.bat`는 EXE 실행이 막히는 환경을 위한 보조 실행 파일입니다. 기본 실행 파일은 `스타샌드 아일랜드 한글패치 런처.exe`입니다.

## v1.0.6 이후 주요 변경점

- Xbox PC `2026.16.54.0` 기준 호환성을 확인했습니다.
- Steam buildid `22932871` 기준 패치를 추가했습니다.
- Steam buildid `22987508` 기준 최신 원문 업데이트를 반영했습니다.
- Steam/Xbox의 원문 차이를 분리해 설치 경로에 맞는 패치가 자동 적용됩니다.
- Steam판 숯가마/용광로/가공 UI에서 `$$UI:...$$` 내부 키가 보이던 문제를 수정했습니다.
- 지원 범위 경고 후 설치가 멈춘 것처럼 보이던 문제를 수정했습니다.
- 일부 환경에서 `The property 'warning' cannot be found` 오류로 설치가 실패하던 문제를 수정했습니다.
- 런처가 오래된 캐시보다 현재 폴더의 최신 패키지를 우선 사용하도록 고쳤습니다.
- 런처 실행 직후 최신 릴리스를 확인하고, 동의 시 다운로드/검증/현재 폴더 갱신/런처 재실행을 처리합니다.
- 다운로드 ZIP의 SHA256 digest 검증을 추가했습니다.
- SUIT soft-weight 폰트 번들을 반영했습니다.
- `FishTank` 계열은 `수족관`, `FishPond` 계열은 `어항`으로 이름을 정리했습니다.
- `Animal` 탭 영어 잔재를 `동물`로 수정했습니다.
- Steam 최신 원문에서 추가된 데이트/동행 대사와 신규 문구를 번역했습니다.
- 파스텔 메일, 솔라라 데이트 대사처럼 캐릭터 말투가 흔들리던 부분을 추가 폴리싱했습니다.
- 여러 인명이 이어지는 목표문에서 조사가 반복되던 표현을 정리했습니다.

## 복원 방법

- GUI 런처의 `원본 복구` 버튼 또는 `Restore_Original_589829.bat`를 실행하면 텍스트 번들과 폰트 번들을 원본으로 복원합니다.

## 특징

- 완성된 게임 번들 전체를 배포하지 않습니다.
- 설치 시 사용자 PC의 로컬 `589829.ab`에 번역 JSON을 직접 주입합니다.
- 게임의 `English` 언어 슬롯을 한글로 덮어쓰는 방식입니다.
- Steam/Xbox 경로 자동 탐지를 지원합니다.
- Steam/Xbox 플랫폼별 텍스트 패치 파일을 분리해 적용합니다.
- 폰트는 검증된 번들을 직접 복사합니다.

## 문제 제보

아래 문제가 보이면 댓글 또는 이슈로 제보해 주세요.

- 영어가 그대로 남아 있는 문장
- `$$UI:...$$` 같은 내부 키 노출
- 문맥이 어색한 번역
- 캐릭터 말투/호칭 불일치
- 아이템명/설계도명 불일치
- UI 줄넘침 또는 잘림
- 설치 실패 또는 런처 오류

제보할 때 Steam/Xbox 여부, 패치 버전, 스크린샷, 가능하면 원문/현재 번역문을 함께 알려주시면 확인이 빠릅니다.

## 안내

- 비공식 팬 패치이며 개발사/퍼블리셔와 무관합니다.
- 게임 원본 파일 및 권리는 각 권리자에게 있습니다.
- 게임 업데이트로 번들 구조가 바뀌면 새 패치가 필요할 수 있습니다.
- 설치 후 게임 내 언어는 반드시 `English`를 선택해야 합니다.
- 상업적 판매 및 유료 배포를 금지합니다.
