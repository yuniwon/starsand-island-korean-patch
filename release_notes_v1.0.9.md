# 스타샌드 아일랜드 한글패치 v1.0.9

이번 버전은 Steam/Xbox 설치본을 자동으로 구분해 알맞은 한글 패치를 적용하는 배포본입니다.
기존 v1.0.6 이하를 사용 중이라면 최신 버전으로 업데이트해 주세요.

## 이번 버전에서 고친 것
- Steam 2026-04-24 업데이트 이후 숯가마/용광로 같은 가공 UI에서 `$$UI:UIQueueProcessView/...$$` 문구가 그대로 보이던 문제를 반영했습니다.
- Steam용 새 제작/가공 UI 문구를 별도 패치 파일에 포함했습니다.
  - 예: `가공 시간:`, `필요`, `획득처:`, `가격:`, `모두 수령`
- GUI 런처에서 지원 범위 경고가 나온 뒤 설치가 멈춘 것처럼 보이던 문제를 수정했습니다.
- 경고 메시지는 표시하되, 숨은 `y/N` 입력 없이 설치가 계속 진행되도록 바꿨습니다.

## 왜 문제가 생겼나요?
- Steam 0.4.9647 계열 업데이트에서 제작/가공 UI가 새 키 묶음인 `UIQueueProcessView`를 사용하도록 바뀌었습니다.
- 이전 패치 파일에는 이 Steam 전용 키가 없어서, 게임이 번역문 대신 내부 키 이름을 그대로 화면에 표시했습니다.
- v1.0.9 배포본에는 Steam용 키와 Xbox용 키를 따로 넣어, 설치 경로에 맞는 파일을 자동으로 적용합니다.

## 설치/업데이트 방법
- 기존 런처를 실행하면 최신 stable 릴리스를 확인하고 다운로드 후 설치할 수 있습니다.
- 직접 설치할 경우 ZIP을 풀고 `스타샌드 아일랜드 한글패치 런처.exe` 또는 `Start_Korean_Patch.bat`를 실행하세요.
- 설치 후 게임 내 언어는 반드시 `English`로 선택해야 한글이 표시됩니다.
