# 스타샌드 아일랜드 한글패치 v1.0.15

v1.0.15는 Steam 2026-04-29 원문 업데이트 대응과 캐릭터 대사톤 추가 폴리싱을 반영한 배포본입니다.

## 이번 버전에서 고친 것
- Steam 최신 원문에서 새로 추가된 데이트/동행 대사와 신규 문구를 번역했습니다.
- Steam 지원 buildid 기준을 최신 설치본 `22987508`로 갱신했습니다.
- 추가 대사 중 직역투가 강하거나 한국어로 어색한 표현을 다시 다듬었습니다.
- 파스텔 메일에서 반말로 남아 있던 문장을 상냥한 존댓말 기준으로 정리했습니다.
- 솔라라 데이트 대사에서 `시작해요!`, `준비됐나요?`, `네, 약속이에요!`처럼 남아 있던 존댓말을 솔라라 반말 톤으로 수정했습니다.
- `가스트와 진저와`, `진저와 루넬과`, `진저와 솔라라와`처럼 이름이 이어질 때 조사가 어색하게 반복되던 목표문을 정리했습니다.
- 최신 Steam/Xbox 플랫폼별 텍스트 패치 파일을 다시 생성해 배포본에 반영했습니다.

## 설치/업데이트
- 기존 런처를 실행하면 새 stable 릴리스 `v1.0.15`를 감지합니다.
- 업데이트에 동의하면 ZIP 다운로드, SHA256 digest 검증, 현재 폴더 갱신, 런처 재실행까지 자동으로 진행됩니다.
- 새로 받는 경우에는 ZIP을 풀고 `스타샌드 아일랜드 한글패치 런처.exe`를 실행하면 됩니다.
- 설치 후 게임 내 언어는 반드시 `English`로 선택해야 한글이 표시됩니다.
