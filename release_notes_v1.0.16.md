# 스타샌드 아일랜드 한글패치 v1.0.16

v1.0.16은 2026-04-30 Steam/Xbox 업데이트 대응과 최근 캐릭터 말투/번역 폴리싱을 반영한 배포본입니다.

## 이번 버전에서 고친 것
- Xbox/Game Pass `2026.18.58.0` 기준을 지원 범위에 반영했습니다.
- Steam `buildid 23021933` 기준을 지원 범위에 반영했습니다.
- Xbox live 589829 원문이 Steam 최신 원문과 동일한 것을 확인하고, Xbox/Steam 모두 최신 번역 working set 기준으로 다시 패키징했습니다.
- 2026-04-30 공식 업데이트 자체에는 새 번역 대상 문장이 없음을 확인했습니다.
- 2026-04-30 저녁까지 진행한 캐릭터 말투/문장 폴리싱을 배포본에 반영했습니다.
- 제린 관계 장면, 관광객 인사/선물 계열, 스텔라 의뢰 완료 대사, 제피리아 모험/의뢰 계열처럼 실제 화면 제보로 확인된 말투 흔들림을 정리했습니다.
- 파스텔 메일, 솔라라 데이트 대사, 다중 인명 목표문 조사 정리 등 v1.0.15의 번역 수정분도 유지했습니다.

## 설치/업데이트
- 기존 런처를 실행하면 새 stable 릴리스 `v1.0.16`을 감지합니다.
- 업데이트에 동의하면 ZIP 다운로드, SHA256 digest 검증, 현재 폴더 갱신, 런처 재실행까지 자동으로 진행됩니다.
- 새로 받는 경우에는 ZIP을 풀고 `스타샌드 아일랜드 한글패치 런처.exe`를 실행하면 됩니다.
- 설치 후 게임 내 언어는 반드시 `English`로 선택해야 한글이 표시됩니다.
